Современная веб-система для автоматизации учёта воспитанников и управления образовательными программами.  
Реализована на платформе **.NET 10** с использованием **Blazor Server**, **Entity Framework Core** и **Docker**.

---

## ✨ Возможности

-  **Управление детьми** — добавление, редактирование и просмотр списка воспитанников с валидацией данных
- 📚 **Каталог курсов** — создание и редактирование образовательных программ с указанием квот мест
- 📝 **Онлайн-запись** — регистрация ребёнка на выбранный курс с автоматической проверкой доступности
-  **Контроль наполняемости** — предотвращение "овербукинга" через серверную валидацию лимитов групп
- ✅ **Валидация FluentValidation** — декларативные правила проверки для всех сущностей
- 🐳 **Контейнеризация** — multi-stage сборка Docker-образа с сохранением данных SQLite через Volume
- 🔄 **Unit of Work** — атомарные транзакции при одновременном изменении нескольких таблиц
- 📊 **Автоматическая инициализация** — применение миграций и seed-данных при первом запуске
